## Getting Started

To run locally on your machine, follow these steps:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/U-Shashank/Task-KampKode
   ```

2. **Navigate to Project Directory**
   ```bash
   cd Task-KampKode
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Run the App**
   ```bash
   npm run dev
   ```
